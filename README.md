# cpuhogtd (CPU Hog Thread Dump)

## config.json

```
{
  "CPU_THRESHOLD": 50.0,
  "CHECK_INTERVAL_MS": 60000,
  "AGGRESSIVE_INTERVAL_MS": 10000,
  "CONCERN_THRESHOLD": 12
}
```

## Usage

`-javaagent:path/to/java-agent-sample-1.0-SNAPSHOT.jar`
